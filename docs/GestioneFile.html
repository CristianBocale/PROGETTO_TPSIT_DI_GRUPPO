GestioneFile()

new GestioneFile()

Questa classe gestisce la lettura e la scrittura di un file. Il file viene utilizzato per memorizzare la lista della spesa in formato JSON. La classe fornisce metodi per leggere i dati dal file e creare una mappa dalla stringa JSON letta.
